written by UDN on 2019-10-01

宜蘭南方澳跨海大橋斷裂，內政部長徐國勇跟11點多到場了解狀況，由於還有船員困在被壓垮的船艙內，部長要專家評估如何搶救，現場遭漁民怒嗆「船主最清楚狀況」，等到專家來，救人太晚了！

宜蘭縣南方澳跨港大橋今天上午斷橋意外，造成多人受傷，南方澳漁港也因此封港，好端端的大橋怎麼會斷裂？地方人士懷疑是去年海堤修復，載送大量消波塊，壓壞大橋。對此，基隆港務分公司主任秘書林健明表示，消波塊載重應在容許範圍內。
