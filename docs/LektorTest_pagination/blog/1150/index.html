written by UDN on 2019-10-01

宜蘭南方澳大橋今天發生斷橋，南方澳文史工作者廖大慶說，斷橋不應賴給颱風，這次颱風南方澳風雨沒有很大，地方人士懷疑，承載負荷太大，這座橋已有20多年歷史，地方人土就曾反映，平常行人走在橋上時，只要車子經過，都會搖晃得很厲害。

鑽研橋梁工程的中央大學土木系教授王仲宇說，該橋以兩側吊索撐住橋面，他看新聞照片，推估是吊索和橋面連接處遭鏽蝕，再加上近日颱風風力導致大力晃動，因而倒塌。他也說，倒塌不是一兩天造成的，主事單位應盡速以空拍機、光達掃描和水下攝影紀錄斷裂情況，也應檢討是否落實政府規定每兩年要做一次的橋梁檢查。
