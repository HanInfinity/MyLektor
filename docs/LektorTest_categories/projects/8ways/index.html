不想被誤認成中年大叔 啤酒肚消風8個妙招再累都得學!

啤酒肚其實就是所謂的「中央肥胖」，其成因是過多脂肪組織附著在你的腹部、胃腸周圍，造成內臟脂肪過高，而內臟脂肪過高的原因除了體重過重之外，也可能是飲食及作息所造成的，例如熬夜、喜歡吃肉及油炸的食物、喜歡喝酒、不愛喝水。不得不說，挺著一顆肥肚，不只穿衣緊繃線條不優，脫衣也超沒看頭，而且超容易年紀輕輕就自動增齡為大叔，當然要想辦法消風才行。運動營養師和中醫師向我們推薦了這幾招減肚妙招，乖乖照著做，一個月減個4~8公分很有機會。 https://www.gq.com.tw/life/health/content-38360.html
